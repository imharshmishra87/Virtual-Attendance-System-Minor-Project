# Virtual-Attendance-System-Minor-Project
Real-time face recognition system powered by InsightFace and OpenCV.<br>

<p>This is the virtual attendance based management system and it mainly involves Collecting the data using insight face built in retina face module and detection using insight's face embedding matchings. This document will help you and will provide you an enough overview of this project</p>

<h2>Tech Stack Used</h2>
<p>mainly Used The following Tech stack for the development of this project</p>
<ul>
<li>Python -- For Writing Code</li>
<li>Open cv -- For Reading images and image manipulation</li>
<li>Insight Fcae-- For Face Recognition and Face detection</li>
</ul>
